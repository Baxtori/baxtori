# Botanical source plates

The Trail reader uses two public-domain nature prints from *The Ferns of Great Britain and Ireland* (1855–1857), edited by John Lindley, with specimens nature-printed by Henry Bradbury.

- `male-fern-nature-print.png` — *Dryopteris filix-mas* (Plate XIV). Source: [Wikimedia Commons](https://commons.wikimedia.org/wiki/File:Dryopteris_filix-mas_Moore14.png).
- `bracken-nature-print.png` — *Pteridium aquilinum* (Plate XLIV). Source: [Wikimedia Commons](https://commons.wikimedia.org/wiki/File:Pteridium_aquilinum_Moore44.png).

Both Commons file pages identify the faithful reproductions of the nineteenth-century works as public domain. The files in this directory preserve the original scan pixels; the reader applies cropping, blending, and scroll-linked reveal effects in CSS.
